Paper ECG

Published September 1, 2020 â€” Updated November 14, 2020

This application provides tools for cardiology professionals to digitize digital scans or images of ECG print outs, and is the first free and open source application to provide such functionality. Users can annotate regions of the image to extra signal data and specify the scale information to create an accurate digitization of the signals. The digitization processes are implementations of peer-reviewed ECG signal digitization algorithms. The results of validating the digitization process on images with ground-truth labels will be published, and the application will be available on Github and PhysioNet.

I created the initial panel layout, built a wrapper library based on PyQt to allow building interfaces via declarative syntax, and implemented the image processing algorithms.
